# 项目功能范围

## 结论

本项目按“资料可信度 + 数据模型 + 计算核心 + 攻略输出”初始化。当前版本已经具备可运行前端、官方唯一装备词缀种子、BD 大厅、独立 BD 详情页、装备详情卡和三赛季 150 层参考矩阵；全量装备库、技能系数、巅峰盘精确节点、Boss 掉落和真实榜单校准是下一阶段需要继续导入和实现的模块。

## 已初始化模块

- 版本与来源治理：区分 2026-06-28 当前在线 `3.0.4` 和 2026-06-30 即将上线 `3.1.0`。
- 职业覆盖：野蛮人、德鲁伊、死灵法师、圣骑士、游侠、法师、魂灵师、术士；8 职业基线来自 2026 年官方《Lord of Hatred》资料片页面。
- 赛季开荒：每个职业都有初始开荒决策规则。
- 装备模型：部位、词缀分类、词缀到计算角色的映射。
- 官方唯一装备种子：从 3.1.0 补丁页抽取 278 条固定词缀数据，并生成装备检索库。
- 伤害计算：期望值模型和可读拆分。
- 配装评分：按流派权重对词缀做基础评分。
- 静态前端：`Harris‘s Diablo 4` 首屏、BD 大厅、独立 BD 详情页、职业开荒、伤害计算、装备详情检索、150 层参考和来源面板。
- 三赛季 BD 档案：按冲层、速刷、日常三个用途输出 243 套结构化 BD，包含装备槽位、可替换件、核心暗金/威能、技能加点顺序、巅峰点击顺序、打法循环、变体和 150 层速度参考。当前已有 30 套 S14 社区来源覆盖：21 套同赛季社区参考、9 套跨赛季社区参考，覆盖野蛮人溶解旋风、德鲁伊震波粉碎/音速撕裂、死灵星界骨魂/终极血浪、游侠飞刀舞/穿透射击、法师连锁闪电/烈焰电荷弹、魂灵师风暴羽毛闪避。

## 应补全模块

- 全量普通/传奇/唯一/神话唯一装备库。
- 全量传奇威能、暗金特效、套装/护符/赛季机制。
- 职业技能、技能标签、技能系数和资源消耗。
- 巅峰盘、雕文、节点路径和等级阶段推荐。
- Boss 掉落表、地下城、梦魇词缀、活动收益。
- 更完整的构筑编辑器、装备对比器、中文攻略详情页、真实榜单回填和机器人查询。

## 完成标准

- `npm run preflight` 通过。
- `npm run fetch:patch` 可重新生成官方唯一装备种子。
- `npm run verify` 覆盖职业、版本、来源、生成数据和代表性伤害样例。
- 所有非官方事实都能追溯到来源，并标记可信度或待验证状态。
